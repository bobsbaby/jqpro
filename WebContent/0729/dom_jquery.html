dom 객체와 jquery 객체 반환

jquery를 dom으로 반환 : jquery객체.get[N] / jquery객체[N]
dom 객체를 jquery객체로 반환 : $(dom 객체 )

jq = $('h1');
jq.css('color', 'blue');


-jquery 객체를 dom으로 변환
dom = jq.get(2); //jq[2]
dom.style.color = 'red';
-dom 객체를 jquery로 변환
$(dom).css('color','green');
-전체 h1 중에서 세번째 선택 : jquery
jq3 = $('h1:eq(3)') // h1 선택자 사용시 $ 사용
jq3.css('color', 'hotpink');

body -> div -> h2, p, button
body의 후손 -> body p(body의 후손 중 p태그)
div > button (div의 자식 중 button)
h2 + p -> h2 바로 뒤에 있는 p태그 (1개만 선택) 
h2 ~ p -> h2의 형제 중 모든 p태그 선택 (다중)-> h2의 위치 다음에 있는 p 태그만 선택된다

E1 E2 -> E1 의 자식 요소 중 모든 E2 요소 (후손)
E1 > E2 -> E1의 바로 아래 자식 요소 중 모든 E2 요소 (자식)
E1 + E2 -> E1의 바로 다음에 나오는 형제요소 중 E2 요소
E1 ~ E2 -:> E1의 다음에 나오는 형제요소 중 모든 E2 요소
