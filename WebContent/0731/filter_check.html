:checked

- $('input[type=checkbox]:checked')
- $(':checkbox:checked')
- 선택한 요소를 대상으로 반복문을 실행한다.
- 반복문 안에서 현재 실행 대상이 되는 객체를 this로 받는다.
- val -> getter, setter역할을 하며 form 하위 객체에서만 사용 가능
- attr -> 속성명과 값으로 설정되어 있음, prop -> true, false : 속성에서 값을 가져 올때 사용
취 미 : 여행 장기 바둑 독서 낚시
